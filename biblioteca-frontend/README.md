# biblioteca-frontend
Um sistema de front-end de gerenciamento de livros para bibliotecas

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
